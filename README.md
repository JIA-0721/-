# 实时多模态AI语音助手
一个基于端云协同架构的实时多模态 AI 对话应用。应用能够同步调用用户的摄像头与麦克风，赋予 AI “看”与“听”的能力，并实现毫秒级的流式语音与视觉闭环交互。项目在追求极致流畅度的同时，针对端云传输和模型算力进行了深度的成本控制优化。
## 核心特性
- 视觉能力：AI能够通过实施摄像头来理解镜头中的环境，任务和事件，然后结合上下文恰当回应
- 智能打断：通过引入端侧VAD，AI 发声时用户可随时开口打断，对话节奏如人类般自然
- 端云成本控制：通过让AI自动识别任务的难度，给予AI自行选择执行任务的大模型的能力

## 成本控制优化策略
- 端侧本地 VAD 过滤： 运行 [如 Silero VAD] 算法，仅在检测到人声时向上行链路发送音频，避免空白噪音空耗流量。
- 视觉动态帧率（Frame-Skipping）： 采用图像帧差法。画面静止时，视频流降底至 1fps；检测到画面突变（如物体移动）时自动提升至 5fps，最大化节省视觉 Token。
- 模型路由与滑窗： 云端核心采用主流高性价比的 [如 Gemini 2.0 Flash / GPT-4o-mini]，并对历史视觉上下文实施动态滑窗清理，防止 Token 膨胀。
